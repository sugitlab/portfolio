# このブログは

このブログは [sugit](https://twitter.com/sugitlab)が常々思っていた、「あ〜、雑に書いたメモでも公開してしまいたい」という、見せたいのか見せたくないのかがハッキリしないモチベーションによって作られました。まじめに書いたものは Qiita や Zenn などに投稿しますので、ここはちょっと実験的なメモや思いついたことを気軽に残せるような場所になる予定です。

## 技術的なこと

このブログは Next.js で作られています。以下の通りとてもシンプルな構成です。

- Next.js
- Tailwind CSS
- HeadlessUI
- Markdown Parser (by Zenn)

## やりたいこと

- Analytics は入れておきたい
- Gist/Dartpad の埋め込み対応

自分で読むときにイライラしそうなので広告はいれません（宣言）
:::
